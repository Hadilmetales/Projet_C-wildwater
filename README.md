# Projet C-Wildwater

Projet de traitement de données d'un système de distribution d'eau.
Réalisé dans le cadre du projet pré-ING2 2025-2026.

## Description
Ce projet analyse un fichier CSV décrivant le réseau d'eau (Sources, Usines, Stockages) et produit des histogrammes sur la capacité et la consommation des stations.

Le projet utilise un `Makefile` pour la compilation.
