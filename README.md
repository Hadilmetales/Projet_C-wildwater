Cursus : Pré-ING2 (2025-2026)

## Description
Ce projet a pour but d'analyser un vaste ensemble de données décrivant un réseau de distribution d'eau en France (Sources, Usines de traitement, Stockages, Consommateurs).

L'application est composée d'un script Shell qui pilote un programme en langage C pour filtrer les données, effectuer des calculs via une structure d'arbre AVL, et générer des histogrammes avec Gnuplot.

## Prérequis
Pour faire fonctionner le projet, vous avez besoin de :
- `gcc` (Compilateur C)
- `make` (Utilitaire de compilation)
- `gnuplot` (Pour la génération des graphiques)

## Compilation
Le projet utilise un **Makefile** pour automatiser la compilation.
Pour construire l'exécutable, lancez simplement la commande :

```bash
make
